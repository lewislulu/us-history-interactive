# 大明王朝 · 兴亡三百年

互动式明代历史人物时间线可视化，涵盖从元末群雄逐鹿到1644年甲申之变的完整历史。

## 预览

基于 D3.js + GSAP 构建的电影级暗色主题时间线，支持：

- **8个章节**：元末群雄、洪武立国、靖难之役、永乐盛世、土木之变、心学与抗倭、张居正改革、明末覆灭
- **15位关键人物**：朱元璋、马皇后、刘伯温、徐达、朱棣、郑和、于谦、王阳明、戚继光、张居正、万历帝、魏忠贤、袁崇焕、崇祯帝、李自成
- **24个重大事件**：每个事件含背景/经过/结果叙事 + 戏剧化对话场景
- **人物传记模式**：聚焦单一人物的生平时间线
- **章节筛选**：按历史阶段浏览
- **缩放与平移**：自由探索时间线

## 技术栈

- [Vite](https://vitejs.dev/) - 构建工具
- [D3.js](https://d3js.org/) - SVG 时间线渲染
- [GSAP](https://greensock.com/gsap/) - 动画
- [Marked](https://marked.js.org/) - Markdown 解析
- Vanilla JavaScript - 无框架依赖

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:5173` 即可体验。

## 构建

```bash
npm run build
npm run preview
```

## 项目结构

```
data/
  timeline.json          # 主配置（时代、章节、角色、事件索引）
  characters/*.json      # 15个人物数据
  events/*.json          # 24个事件元数据
  events/*.md            # 24个事件叙事（背景/经过/结果 + 场景对话）
src/
  main.js               # 入口
  core/                  # 时间线引擎、路由、小地图
  ui/                    # UI组件（章节栏、人物卡、事件面板等）
  data/                  # 数据加载与Markdown解析
  styles/                # 样式
  utils/                 # 工具函数与动画
```

## 操作说明

- **点击人物头像** → 查看人物卡片
- **点击事件节点** → 查看事件详情
- **点击场景卡片** → 进入对话场景
- **按 R 键** → 重置缩放
- **按 Esc 键** → 关闭面板
- **鼠标滚轮** → 缩放
- **拖拽** → 平移
